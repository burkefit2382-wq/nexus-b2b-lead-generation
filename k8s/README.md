# NEXUS AKS Manifests

These manifests deploy the Dockerized NEXUS API to AKS.

The default image points to the Azure Container Registry name generated by this repo's Azure Bicep:

```text
acrnexusprodnexus.azurecr.io/nexus-api:latest
```

If your build pipeline publishes a different image name or tag, update `k8s/deployments/nexus-api-deployment.yaml` or `helm/nexus/values.yaml`.

## Secrets

Do not commit real production secrets. The real secret file is ignored by Git:

```text
k8s/secrets/nexus-secrets.yaml
```

Create it from the template:

```powershell
Copy-Item k8s/secrets/nexus-secrets.template.yaml k8s/secrets/nexus-secrets.yaml
```

Then replace every `REPLACE_ME` value locally.

Apply manually:

```bash
kubectl apply -f k8s/namespaces/nexus-namespace.yaml
kubectl apply -f k8s/secrets/nexus-secrets.yaml
kubectl apply -k k8s
```

For GitOps, prefer Azure Key Vault + External Secrets Operator or Sealed Secrets instead of committing raw `Secret` values.
