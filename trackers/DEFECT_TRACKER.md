## Severity Definitions

| Severity | Definition | Launch Rule |
| --- | --- | --- |
| Critical | Security bypass, data leak/loss, auth/RBAC failure, billing error, unavailable core product | Must fix before launch |
| High | Core workflow broken, misleading enrichment/scoring, major performance issue | Must fix or explicitly limit launch |
| Medium | Important issue with workaround | Triage before launch |
| Low | Minor polish, copy, or cosmetic issue | Can defer |
